📝 Registration Form

A simple registration form built using HTML.
This project collects user details like name, email, password, etc.

📁 Project Structure
project/
│── index.html
│── style.css
│── script.js
│── README.md
🛠️ Technologies Used

HTML5

CSS3 (for styling)

JavaScript (for validation, optional)

🚀 How to Run

Download or clone the project

Open the folder

Open index.html in any browser

📋 Features

User registration form

Input fields (Name, Email, Password, etc.)

Simple and clean design

Easy to customize

📌 Form Fields Included

Full Name

Email Address

Password

Confirm Password

Gender (optional)

Submit Button

💡 Purpose

This project is useful for:

Beginners learning HTML forms

School or college assignments

Practice for web development

⚠️ Note

This is a frontend-only project

No data is stored (no backend connected)
